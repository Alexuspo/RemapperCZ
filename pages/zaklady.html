Základy ReMapper

Vytvoření projektu


import { map } from "remapper";

map.setSettings({
    bpm: 120,
    noteJumpSpeed: 16
});

Základní objekty

Notes (Kostky)


const note = new Note(0);  // Vytvoření noty v čase 0
note.type = 0;            // Červená kostka
note.direction = 8;       // Směr nahoru
note.push();             // Přidání do mapy

Walls (Zdi)


const wall = new Wall(0, 4);  // Zeď od času 0 do 4
wall.scale = [1, 1, 1];      // Velikost zdi
wall.push();

Animace

ReMapper používá systém keyframů pro animace:


objekt.animate([
    [0, [1, 1, 1]],     // Počáteční stav
    [1, [2, 2, 2]]      // Konečný stav
], "scale");            // Animovaná vlastnost
